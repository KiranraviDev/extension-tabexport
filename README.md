# TabExport Pro - Readme Documentation

## Overview
TabExport Pro is a lightweight browser extension that helps you quickly export your current browser tabs into useful file formats for saving, sharing, or documenting work.

## What the extension does
- Exports open tabs to Markdown
- Exports open tabs to CSV
- Exports open tabs to plain Text
- Works locally in the browser without requiring sign-in or cloud storage

## How to use it
1. Open the extension popup from the browser toolbar.
2. Enter a project name if needed.
3. Click one of the export buttons:
   - Export Markdown
   - Export CSV
   - Export Text
4. The file will be downloaded to your computer.

## Suggested use cases
- Saving research tabs for a project
- Sharing tab lists with teammates
- Creating a simple checklist or report from open tabs
- Keeping a record of browsing sessions

## Notes
- The extension uses the browser tab permission to read the currently open tabs.
- Exported files are generated locally on your device.
- The popup interface is intentionally simple and fast to use.

## Installation in browser
1. Open your browser’s extensions page.
2. Enable Developer Mode.
3. Load the extension folder.
4. Pin the extension to the toolbar for quick access.
